Meditation Cube Firmware

The Meditation Cube Firmware project, led by me, aimed to create the firmware for a meditation aid device known as the "Meditation Cube." This innovative project involved the use of various hardware components and programming techniques to create a unique and interactive meditation experience. Key highlights of the Meditation Cube Firmware project include: - **Encoder Control**: We implemented precise encoder control to allow users to adjust meditation settings easily. The firmware ensures a smooth and responsive user interface. - **LED Visualization**: The Meditation Cube features LED lights that synchronize with the user's breathing rhythm, creating a calming visual experience. The firmware controls the LED patterns to enhance meditation sessions. - **Breathing States**: The firmware manages different meditation states, including inhale, exhale, and hold. It provides accurate timing and transitions to guide users through their meditation practice. Throughout the development process, the project received positive feedback from the team and contributed to the creation of a functional and user-friendly Meditation Cube. This project showcased my expertise in firmware development and my ability to work on innovative hardware projects.
